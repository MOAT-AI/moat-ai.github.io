# MOAT Collaboration GitHub Pages Site

This repository contains the **Jekyll** source for the MOAT Collaboration homepage.

## Content Structure

- `index.md` defines the homepage metadata
- `_sections/*.md` contains the homepage sections in markdown
- `_layouts/home.html` assembles those markdown sections into the landing page
- `assets/css/site.css` provides the original site styling

## Homepage Sections

1. Introduction
2. Participants
3. How To Get Involved
4. AI-Enabled Ecosystem

The site now includes a first-pass public narrative for these sections based on the MOAT collaboration source material in `Vision/Starting_point/`.
